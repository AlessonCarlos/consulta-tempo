# Weather App

Estrutura inicial do projeto de exemplo para um aplicativo de clima.

Estrutura de pastas:

/
├── index.html
├── README.md
├── assets/
│   ├── icons/weather/
│   └── images/
├── css/
│   ├── reset.css
│   └── styles.css
├── js/
│   ├── api/weatherService.js
│   ├── ui/renderWeather.js
│   ├── utils/helpers.js
│   └── main.js
└── .gitignore

Como usar:
- Abra `index.html` em um servidor local ou no navegador.
- Implemente a integração com a API de clima em `js/api/weatherService.js`.
